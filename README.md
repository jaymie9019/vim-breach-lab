# Vim Breach Lab

一个 macOS 优先的 Node.js 终端训练营，用系统自带的 `vim` 来跑关卡。重点不是模拟 Vim，而是把真实 Vim 放进受控沙盒里练，退出后给你做判题、评分、diff 和复盘。

## 运行

```bash
npm start
```

启动后会进入终端界面。选择关卡后，程序会：

1. 在临时目录生成练习文件
2. 用隔离参数启动系统 `vim`
3. 让你在真实 Vim 里完成任务
4. 退出后做结算和复盘

建议用 `ZZ` 结束关卡，这样步数统计会自动去掉退出按键。

## 浏览器原型

旧的 Web 原型仍然保留在仓库里作为样机：

```bash
npm run web:start
```

## 测试

## 当前范围

- 单人终端训练营
- 46 个关卡，覆盖 `f`、`w`、`t`、`%`、`ciw`、`ci"`、`da(`/`dab`、`.`、`x`、`dw`、`cw`、`u`、`^`、`$`、`g_`、`A`、`I`、`o`、`O`、`C`、`D`、`J`、`/`、`?`、`n`、`N`、`*`、`#`、`v`、`V`、count 前缀
- 临时沙盒文件
- 真实 `vim` 子进程集成
- 通关判定、diff、推荐解法、本地进度记录
- 基于按键日志的近似步数统计
- 可选关卡技法要求，例如“本关至少使用一次 `u`”

## 已知限制

- 第一步只支持 macOS 优先场景
- 步数统计基于 `vim -W` 的按键日志和退出命令归一化，不是完整的编辑器遥测
- 浏览器原型仍在仓库中，但不再是主线产品
