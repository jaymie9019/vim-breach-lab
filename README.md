# Vim Breach Lab

一个 macOS 优先的 Node.js 终端训练营，用系统自带的 `vim` 来跑关卡。重点不是模拟 Vim，而是把真实 Vim 放进受控沙盒里练，退出后给你做判题、评分、diff 和复盘。

## 运行

```bash
npm start
```

启动后会进入终端界面。选择关卡后，程序会：

1. 在临时目录生成练习文件
2. 用隔离参数启动系统 `vim`
3. 让你在真实 Vim 里完成任务
4. 退出后做结算和复盘

建议用 `ZZ` 结束关卡，这样步数统计会自动去掉退出按键。

## 当前 TUI 体验

现在的主界面不是“游戏大厅”路线，而是更偏学习工具：

- 课程地图：先回答“我现在学到哪了”
- 章节讲义：先建立这一章的心智模型
- 关卡说明：开始前把目标、光标和练法看清楚
- 结算复盘：先指出最先出错的地方，再给下一步建议

主线学习路径是：

1. 在课程地图里定位当前章节和关卡
2. 先读章节讲义，再进单关
3. 在真实 `vim` 里完成任务
4. 回到复盘页看文本、光标、技法和步数

设计上刻意避免做成 dashboard，而是更像“教练带练”：

- 先讲判断，再讲动作
- 先做对，再做漂亮
- 先修最先出错的地方，再谈优化

如果后续继续改 TUI，建议优先保持这几个原则，不要退回到信息很满、边框很多、像控制台报表的风格。

## 当前范围

- 单人终端训练营
- 66 个关卡，覆盖 `f`、`w`、`t`、`%`、`ciw`、`ci"`、`da(`/`dab`、`.`、`x`、`dw`、`cw`、`u`、`^`、`$`、`g_`、`A`、`I`、`o`、`O`、`C`、`D`、`J`、`/`、`?`、`n`、`N`、`*`、`#`、`v`、`V`、`"0`、`"_`、`"a`、`"A`、`"b`、`p`、`P`、`ma`、`` `a ``、`'a`、`''`、`'.`、`gi`、`gv`、`g;`、`guw`、`gUw`、`g~w`、`:s`、`:%s`、`:g`、`:norm`、count 前缀
- 临时沙盒文件
- 真实 `vim` 子进程集成
- 通关判定、diff、推荐解法、本地进度记录
- 基于按键日志的近似步数统计
- 可选关卡技法要求，例如“本关至少使用一次 `u`”

## 已知限制

- 第一步只支持 macOS 优先场景
- 步数统计基于 `vim -W` 的按键日志和退出命令归一化，不是完整的编辑器遥测
- 浏览器原型仍在仓库中，但不再是主线产品
